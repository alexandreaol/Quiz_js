Quiz: Operador Ternário em JavaScript

1. Qual é a finalidade do operador ternário em JavaScript?




✅ O operador ternário é uma forma curta de fazer um if-else em uma linha.

2. O que será impresso?
let idade = 20;
console.log(idade >= 18 ? "Maior de idade" : "Menor de idade");




✅ A condição idade >= 18 é verdadeira, então o resultado será "Maior de idade".

3. Qual das opções representa a forma correta de usar o operador ternário?




✅ A sintaxe correta do ternário é: condição ? valor_se_verdadeiro : valor_se_falso.

4. Se nota = 5.5, o que será exibido?
console.log(nota >= 6 ? "Aprovado" : "Reprovado");




✅ Como 5.5 não é maior ou igual a 6, a resposta será "Reprovado".

5. Ternários podem ser usados dentro de template strings?




✅ Template strings aceitam expressões dentro de `${}`, incluindo ternários.
